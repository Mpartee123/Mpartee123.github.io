Wait It Out

A simple web app built to quickly find things to do. Users select from three main categories and three sub categories to find their desired activity. This app was based on the idea that traffic is not a pleasant experience and users would rather find something else to do if it was close by.

Technology

Wait It Out uses Bootstrap3 for a mobile-first layout, with jQuery for easy page manipulation. On the back end the page makes requests to both the Google places API and the Google maps API to pull results that are both close by and open at the time of the request.
