Wait It Out

A simple web app built to quickly find things to do. Users select from three main categories and three sub categories to find their desired activity. This app was based on the idea that traffic is not a pleasant experience and users would rather find something else to do if it was close by.

Technology

Wait It Out uses Bootstrap3 for a mobile-first layout, with jQuery for easy page manipulation. On the back end the page makes requests to both the Google places API and the Google maps API to pull results that are both close by and open at the time of the request.

The Tube

A scheduling web app where users can add information about a train and the next arrival will be calculated from the user inputs.

Technology

The Tube uses both Bootstrap 3 and Jquery. Behind the scenes the app utilizes firebase Moment.JS. Moment in particular is heavily used for time calculation and date validation.

My Cousin Liri

My Cousin Liri is a CLI based app that takes in a user request and responds with information gathered from 3 different APIs.

Technology

M.C.L is a node based app. It makes use of several NPM dependencies such as Moment, Axios, Fs, Node-Spotify-API, and Dotenv. As this is a CLI based app you will need to provide your own .env file with the appropriate keys. More information is available in the README.md on github.
